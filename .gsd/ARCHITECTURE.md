## Overview

Enterprise-Grade AI-Powered Data Quality Assurance Platform. Connects to diverse data sources, validates using AI and custom rules via LangGraph, and presents results in a React-based dashboard. Uses RAG for context management to prevent hallucination.

### Infrastructure
- **Purpose:** Data persistence, caching, and embeddings
- **Location:** Docker network / external

## Data Flow

1. User connects data source via Frontend -> API maps it using appropriate `connector`
2. LangGraph Agent pipeline triggered (Profile -> Validate) using `Validation Engine` and configured `LLM`
3. RAG Service (ChromaDB) supplies context to mitigate AI hallucinations
4. Validation results and metrics saved to PostgreSQL; Real-time status in Redis
5. Frontend polls or queries API for results, building visual dashboards
